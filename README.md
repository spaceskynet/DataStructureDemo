# DataStructureDemo

## 色彩搭配

```cpp
QColor(0, 255, 0, 255), // 空闲，Green
QColor(255, 0, 0, 255), // 已使用，Red
QColor(255, 138, 217),	// 链表
QColor(152, 255, 237),	// 数组
QColor(183, 185, 255),	// 栈
QColor(220, 255, 108),	// 堆
QColor(255, 81, 94), 	// 树
QColor(255, 168, 117),	// 无向图
QColor(255, 240, 53),	// 有向图
```
